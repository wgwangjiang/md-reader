关于

应用名称：Markdown 阅读器

版本：1.0.0

作者：MdReader Team

描述：一个使用 Rust 和 Tauri 构建的 Markdown 文件阅读和编辑器，支持GitHub Flavored Markdown扩展

🛠️ 功能列表

新建文件 新建空白 Markdown 文件并进入编辑模式
打开文件 打开 Markdown 文件（.md、.markdown、.txt）
编辑/预览 切换分屏编辑模式和纯预览模式
查找 在文档中搜索关键词
替换 查找并替换所有匹配内容
字体设置 自定义字体和字号
导出 PDF 将文档导出为 PDF 格式
导出网页 将文档导出为 HTML 网页
链接跳转 点击预览中的链接在默认浏览器中打开

⌨️ 快捷键

Ctrl + N 新建文件
Ctrl + O 打开文件
Ctrl + S 保存文件
Ctrl + F 打开查找面板
Esc 关闭查找/字体/关于面板

💡 提示：在 macOS 上可使用 Cmd 键代替 Ctrl 键
