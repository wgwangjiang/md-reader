软件使用说明

简介：
本应用为 MdReader（Markdown 阅读器）桌面应用，提供在本地打开、浏览、渲染和导出 Markdown 文档的功能。

主要功能：
- 打开本地 Markdown 文件（.md）并渲染为可读格式。
- 支持基础 Markdown 语法：标题、列表、代码块、表格、图片与链接。
- 支持主题切换与字体缩放以改善阅读体验。
- 导出为 PDF 或截图（若应用打包时包含相应功能）。
- 注册/取消注册文件关联（Windows 注册表脚本已包含在项目根目录）。

界面说明：
- 左侧：文件浏览与目录树（可快速切换文档）。
- 中间：文档渲染视图，支持实时预览。
- 右侧/上方：设置与导出按钮（视界面布局而定）。

快速开始（不涉及构建）：
1. 从发布版本（可执行文件）启动应用。若直接使用源码运行，参考项目 README 或联系开发者获取预编译包。
2. 在应用内通过“打开文件”或将 .md 文件拖入窗口来加载文档。
3. 使用视图上方/侧边的控件切换主题、调整字体大小或导出文档。
4. 若要在 Windows 上将 .md 文件与应用关联，运行项目根目录下的 `register-file-assoc.reg`；若要取消关联，运行 `unregister-file-assoc.reg`。注意：修改注册表请谨慎并备份。

常见问题：
- 图片不显示：确认图片为相对/绝对路径且文件存在；若是网络图片，请检查网络连接。
- 中文或字体问题：在设置中选择合适字体或安装系统字体。
- 导出失败：确认当前版本支持导出功能，或检查文件写入权限。

反馈与支持：
